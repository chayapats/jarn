## demo.gif

**Status: not yet recorded — PRE-TAG manual DoD item.**

The demo GIF is generated by running:

```bash
./scripts/record-demo.sh
```

This requires [vhs](https://github.com/charmbracelet/vhs) (`brew install vhs`) and
`gifsicle` (`brew install gifsicle`).  The script runs `demo.tape`, which sets
`JARN_DEMO=1` so the recording is fully deterministic (canned-response provider —
no real API key needed).  Target size: < 3 MB.

Until the GIF is recorded and committed the README image reference will 404 in
preview — that is expected on this branch.  Recording the GIF is item **T-4-8**
on the launch checklist in `RELEASE.md`.
